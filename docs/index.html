Download

Stand-alone

"Astarisx" is exposed as a global variable. Download it. 


    <script src="astarisx.min.js"></script>

Bower


    $ bower install astarisx

CommonJS


    $ npm install astarisx

AMD


    require(['./astarisx.min.js'], function (Astarisx) {
        // Do something with Astarisx
    });
